# Drug Interaction Checker

A simple application to check potential interactions between medications.

## Features
- Search for interactions between two or more drugs
- Get severity levels (Major, Moderate, Minor)
- User-friendly interface

## Tech Stack
- Frontend: React, Bootstrap
- Backend: Node.js, Express
- Database: Drug interaction API (e.g., RxNorm)

## Installation
1. Clone the repo: `git clone https://github.com/your-username/drug-interaction-checker.git`
2. Install dependencies: `npm install`
3. Start the app: `npm start`

## Usage
1. Enter drug names
2. Click "Check Interactions"
3. View results

The Drug Interaction Checker app:
- *Checks interactions*: Identifies potential interactions between 2+ medications.
- *Provides severity levels*: Categorizes interactions as Major, Moderate, or Minor.
- *User-friendly interface*: Easy to input drugs and view results.
- *Accurate data*: Uses APIs like RxNorm for reliable interaction data.
